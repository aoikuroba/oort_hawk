# 项目结构

给出项目文件结构，便于开发与后续维护

```bash
/
├── README.md
├── docs						//文档目录
├── index.html
├── node_module
├── package-lock.json
├── package.json
├── src							//工作目录
├── vite.config.js
└── yarn.lock
```

```bash
/src
├── App.vue					//入口页面
├── main.js					//入口脚本
├── api							//api相关
├── mock						//数据测试
├── router					//路由
├── store						//全局存储
├── styles					//样式文件
├── utiles					//实用程序
├── components			//页面部件
└── views						//页面目录
```
